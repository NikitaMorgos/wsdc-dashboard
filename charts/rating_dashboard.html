🏅 Рейтинг танцоров по дивизионам

WSDC Points Registry · 15.04.2026

По основной лестнице (Newcomer → All-Stars) — одна строка на танцора: самый старший дивизион в реестре (без исторических Novice/Int). Sophisticated и Masters — дополнительно, отдельные вкладки. Участники с 0 очков — по конкурсам DC, если дивизион конкурса не ниже текущего в реестре.
